# Technology Stack

| Layer | Technology | Purpose |
| --- | --- | --- |
| Frontend | Next.js + React | Modern frontend framework with App Router support and reusable component architecture |
| Backend | Next.js API Routes | Lightweight backend API handling within the same application |
| Language | TypeScript | Strong typing, maintainability, and better developer experience |
| UI Framework | CoreUI React Admin Template | Professional dashboard UI components and layouts |
| CSV Processing | PapaParse | CSV parsing and row transformation |
| Charts | Recharts | Dashboard charts and visual analytics |
| AI Insights | OpenAI API / Fallback Mock Insight | Customer-friendly operational insights |
| Hosting | Vercel | Simple deployment and GitHub integration for Next.js |
